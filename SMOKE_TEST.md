# Чек-лист смок-тестирования RouteWise

Перед релизом пройти критические сценарии вручную.

## 1. Регистрация и вход
- [ ] Открыть /register, заполнить форму, отправить
- [ ] Проверить редирект и отображение профиля
- [ ] Выйти, открыть /login, войти по email/пароль
- [ ] Проверить сохранение сессии после перезагрузки

## 2. Главная и лента
- [ ] Главная (/) загружается без ошибок
- [ ] Посты отображаются (или заглушка)
- [ ] Лайк поста работает
- [ ] Комментарий к посту отправляется
- [ ] Создание поста (если есть кнопка) — форма открывается и отправка

## 3. Профиль
- [ ] /profile открывается, данные пользователя видны
- [ ] Редактирование профиля (аватар, имя, био) сохраняется
- [ ] Подписки/подписчики отображаются

## 4. Карта и гео
- [ ] /map открывается, карта рендерится
- [ ] Поиск места работает
- [ ] Сохранение места (если есть) работает
- [ ] Геолокация запрашивается без краша

## 5. Шорты
- [ ] /shorts открывается
- [ ] Видео/карточки отображаются или заглушка
- [ ] Лайк/комментарий (если есть)

## 6. AI-ассистент
- [ ] Виджет ассистента открывается
- [ ] Отправка сообщения не даёт ошибки в консоли
- [ ] Ответ (или заглушка) отображается

## 7. Бронирования
- [ ] /bookings открывается
- [ ] Список бронирований или заглушка
- [ ] Детальная страница бронирования открывается

## 8. Бизнес-кабинет (роль business)
- [ ] Вход под пользователем с role=business
- [ ] В меню есть «Бизнес-кабинет»
- [ ] /business/dashboard открывается, статистика/графики
- [ ] /business/bookings, /business/promotions, /business/settings открываются
- [ ] Создание/редактирование акции (модалка)
- [ ] Редактирование профиля компании (форма, логотип)

## 9. Админ-панель (роль admin)
- [ ] Вход под пользователем с role=admin
- [ ] В меню есть «Админ-панель»
- [ ] /admin/dashboard открывается
- [ ] /admin/users — таблица, поиск, фильтр по роли
- [ ] Бан пользователя (модалка с причиной)
- [ ] Верификация бизнеса
- [ ] /admin/posts — список, удаление/скрытие поста
- [ ] /admin/reports — список жалоб, разрешение (удалить/предупредить/отклонить)

## 10. Настройки
- [ ] /settings доступна только авторизованным
- [ ] Вкладка «Профиль» — форма, сохранение
- [ ] Вкладка «Безопасность» — смена пароля (или заглушка)
- [ ] Вкладка «Уведомления» — чекбоксы, сохранение
- [ ] Вкладка «Язык» — переключение ru/en, интерфейс меняется

## 11. Язык и доступность
- [ ] Переключатель языка (в настройках) меняет тексты на всех страницах
- [ ] Нет хардкода строк вне локалей
- [ ] Консоль браузера без ошибок на основных страницах

## 12. Сборка и деплой
- [ ] `npm run build` завершается без ошибок
- [ ] `npm run test -- --forceExit` — тесты проходят
- [ ] Продакшен (https://routewise.ru) открывается
- [ ] PWA: установка на главный экран (мобильное устройство)
- [ ] Service Worker зарегистрирован (DevTools → Application)

## 13. Мониторинг
- [ ] Sentry: при ошибке в приложении событие уходит (если DSN настроен)
- [ ] Vercel Analytics (если подключён): данные в дашборде

---

*После прохождения отметить выполнение и зафиксировать критические баги.*
